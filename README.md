A food delivery application built with React.js, Node.js, and Express.js. Users can browse through a variety of restaurants, view the menu, add items to their cart, and place an order for delivery.

Features

User Authentication: Register and login to access personalized features.

Browse Restaurants: View available restaurants and their menus.

Add to Cart: Select and add items from the menu to the cart.

Responsive Design: The app is fully responsive and optimized for mobile devices.

Technologies Used

Frontend: React.js, CSS, Axios

Backend: Node.js, Express.js

Database: MongoDB (for storing user and order data)

Authentication: JWT (JSON Web Token)

Other: React Router (for navigation), Redux (for state management)
